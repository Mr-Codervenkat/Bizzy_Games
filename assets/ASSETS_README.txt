REQUIRED ASSETS
===============
Place these files in this folder:

- icon.png         (1024x1024 px, app icon)
- splash.png       (1242x2436 px, splash screen)
- adaptive-icon.png (1024x1024 px, Android adaptive icon)
- favicon.png      (196x196 px, web favicon)

You can generate placeholder icons at:
  https://icon.kitchen/

Or use Expo's default icons by removing icon/splash references from app.json temporarily.
